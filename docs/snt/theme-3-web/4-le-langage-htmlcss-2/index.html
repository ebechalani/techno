CSS

Le langage CSS (pour Cascading Style Sheets) permet de définir le style graphique d’une page Web (arrière-plan, type et taille des polices de caractères, etc.). La modification du style d’un élément dans une feuille de style se répercute automatiquement à toutes les pages HTML qui y font appel.

Retournez sur le site de jsfiddle et copiez/collez le code suivant dans la case CSS. En ayant pris la peine de récupérer votre code html précédent. Appuyez sur Run.

body{

background-color: white;/*ligne 2*/

font-family: "OpenSans", sans-serif;

padding: 5px 25px;

font-size: 18px;/*ligne 5*/

margin: 0;

color: purple;

}

h1{

font-family: "Merryweather",serif;

font-size: 32px;

color: #FF0000;

}

Créer un site Web dans Notepad++

Télecharger notepad++ en appuyant sur ce lien

Les étapes à suivre pour créer une page html dans notepad++:

1- File--> New

2- Appuyez sur l'onglet Language-->H-->Html

3- Sauvegardez votre page avec l'extension .html; exemple: page1.html

Les étapes à suivre pour créer une fichier CSS dans notepad++:

1- File--> New

2- Appuyez sur l'onglet Language-->C-->CSS

3- Sauvegardez votre page avec l'extension .CSS; exemple: style1.css

Pour appeler le css dans notepad++

Ajoutez la balise qui appelle le code css dans html <link rel="stylesheet" href="styles.css">

A noter que href est la référence du fichier css que vous avez créé, si je prends le cas précédent mon fichier css est nommé style1.css alors il faut modifier href ainsi:

<link rel="stylesheet" href="style1.css">

Insertion d'image

Utiliser la balise <img> pour insérer une image .

Supposons qu'une image nommée html5.png soit stockée dans le même répertoire que votre fichier html de travail.

Le code en langage html ci-dessous:

<img src="html5.png">

On peut également modifier la largeur et la hauteur de l'image en ajoutant les arguments: Width et height.

Exemple pou réduire la largeur et la hauteur de l'image précédente à moitié:

<img src="html5.png" width="50%" height="50%>

Insertion d'un lien hypertexte

Un lien hypertexte nous permet d’accéder de nouvelles pages à partir de la page courante en se référant au lien de la nouvelle page .

Avant d'insérer un lien hypertexte vers une page, il faut que la page existe. Pour cela le lien hypertexte sera ajouté une fois que vous ayez terminer et sauvegarder toutes vos page html.

La balise <a href="page2.html">Next</a> crée un bouton Next sur la page qui nous mène vers la page 2.

Aligner un paragraphe

Pour aligner un paragraphe, on utilise la balise <p> qui prend comme argument l' option align="center" / "left"/"right" et qui doit se fermer une fois le paragraphe est terminé </p>

exemple: pour insérer la phrase "bonjour tout le monde au miieu" -->

<p align="center">

bonjour tout le monde

</p>

N.B: vous pouvez également aligner une image en utilisant la balsie <p align="..."> <img src=..."> </p>
