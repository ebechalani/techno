🎮 Les Coulisses du Web

Derrière chaque page web se cache un code secret… aujourd'hui, vous passez côté coulisses.

💾 Tes réponses sont enregistrées automatiquement sur cet appareil.

🎯 Le défi

Vous regardez des pages web tous les jours. Mais vous êtes-vous déjà demandé ce qu'il y a derrière ? Qui écrit ce que vous voyez ? D'où vient la page quand vous tapez une adresse ? Et comment un moteur de recherche retrouve-t-il une page parmi des milliards ? Aujourd'hui, on soulève le rideau : bienvenue dans les coulisses du Web.

🧰 Matériel et organisation

  • Durée : 40 minutes
  • Équipes : un îlot de 4 = une équipe, avec un nom d'équipe (ex. Les Balises Rouges, Team 404…).
  • Rôles : un rapporteur, un coureur PC (seul à aller aux ordinateurs en U), un gardien du temps, un secrétaire.
  • Le professeur prépare : le tableau pour les scores, des ciseaux par îlot, et il note au tableau le nom d'une page du site de la classe à ouvrir en manche 1.

🕹️ Déroulé

Manche 1 — Sous le capot (12 min, PC en U)

Le coureur PC ouvre la page du site de la classe indiquée au tableau, puis appuie sur Ctrl + U (ou clic droit → « Afficher le code source »). Surprise : voilà ce que reçoit réellement votre ordinateur ! Le coureur fait des allers-retours pour rapporter les réponses au secrétaire.

Recopiez un mot ou un morceau de texte de la page que vous retrouvez aussi dans le code source : ……………………………………

Recopiez une « étiquette » du code écrite entre les signes < et > (on appelle cela une balise) : ……………………………………

À votre avis, qui a envoyé ce code à votre ordinateur ? ……………………………………

Barème : +3 pts par réponse correcte, +2 pts bonus si l'équipe repère la balise qui contient le titre de la page.

Manche 2 — Les paires du Web (12 min, à l'îlot)

Découpez les cartes en bas de page : 6 mots, 6 rôles. Retournez ensuite cette feuille face cachée, mélangez les cartes faces cachées au centre de l'îlot et jouez au memory : chacun retourne deux cartes à son tour ; si l'équipe estime que le mot et le rôle vont ensemble, la paire est gagnée ; sinon, on repose les cartes faces cachées. À la fin, recopiez vos associations ci-dessous : elles seront vérifiées lors de la correction en classe entière.

Vos associations : ……………………………………

Barème : +2 pts par paire correcte à la correction (maximum 12 pts).

Manche 3 — Le procès du Web (8 min, à l'îlot)

Débat-éclair : « Le Web et Internet, est-ce la même chose ? » Chaque équipe écrit sa réponse (oui/non) et jusqu'à 3 arguments. Le rapporteur a 30 secondes pour plaider.

Votre verdict et vos arguments : ……………………………………

Barème : +1 pt par argument recevable, +3 pts si le verdict est correct (indice : l'un des deux existait avant l'autre…).

🏆 Score et débrief

Total des trois manches au tableau : l'équipe gagnante est nommée Webmestre d'honneur. Terminez par le Défi éclair en bas de page.

Débrief oral (5 min) :

  • Une page web, c'est du texte, du code… mais qui l'envoie, et qui l'affiche ?
  • Que fait votre navigateur avec le code HTML qu'il reçoit ?
  • Et un moteur de recherche, comment fait-il pour connaître des milliards de pages ?

Autant de mystères que la séquence « Le Web » va résoudre.

✂️ À découper — les paires du Web (manche 2)

Cartes MOT Cartes RÔLE
HTML Le langage qui décrit le contenu de la page (titres, textes, images)
CSS Le langage qui décore la page (couleurs, tailles, mise en page)
Navigateur Le programme qui reçoit le code et affiche la page à l'écran
Serveur web L'ordinateur distant qui envoie la page quand on la lui demande
URL L'adresse unique qui permet de trouver une page
Moteur de recherche Le service qui explore le Web et retrouve les pages pour vous
