Capteurs, actionneurs et interface homme-machine

Percevoir, agir et dialoguer avec l'utilisateur.

💾 Tes réponses sont enregistrées automatiquement sur cet appareil.

Percevoir et agir

Pour interagir avec le monde, un système embarqué utilise :

  • des capteurs : ils mesurent une grandeur de l'environnement et la transforment en information (température, lumière, mouvement, présence…).
  • des actionneurs : ils agissent sur l'environnement à partir d'un ordre (moteur, chauffage, LED, haut-parleur…).

Entre les deux, le programme décide : il acquiert la donnée du capteur, la traite, puis commande l'actionneur.

Dans une véranda connectée : si la température dépasse 26 °C, on ouvre la fenêtre (moteur).

a. Quel est le capteur ? Quel est l'actionneur ?

b. Écris la règle sous la forme « Si … alors … ».

Activité 3 — L'interface homme-machine (IHM)

L'IHM est ce qui permet à l'utilisateur de dialoguer avec l'objet : écran, boutons, application sur smartphone, voix… Elle affiche des informations et reçoit les commandes.

Certains systèmes reçoivent des commandes sous forme d'un code. Par exemple, un robot reçoit "A50" pour dire « moteur A à 50 % ». On utilise un tableau de correspondance entre le code et l'action :

Code reçu Action
A0 Moteur A arrêté
A50 Moteur A à 50 %
A100 Moteur A à fond

a. Que fait le robot s'il reçoit "A100" ?

b. Quel code faut-il envoyer pour arrêter le moteur A ?

c. Cite deux exemples d'IHM que tu utilises au quotidien.

Ce que je retiens

Un système embarqué acquiert des informations par ses capteurs, les traite, puis commande ses actionneurs. L'interface homme-machine (IHM) permet à l'utilisateur de dialoguer avec l'objet (écran, boutons, application, voix). Les commandes peuvent passer par un code associé à chaque action.
