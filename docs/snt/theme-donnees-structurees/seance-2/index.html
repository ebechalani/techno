Traiter les données : trier, filtrer, croiser

Rechercher, trier, filtrer et calculer sur des données, au tableur et en Python.

💾 Tes réponses sont enregistrées automatiquement sur cet appareil.

Activité 1 — Rechercher, trier, filtrer

Une fois les données rangées en table, on peut les traiter :

  • Rechercher : retrouver un objet précis.
  • Trier : ranger les objets dans l'ordre (croissant ou décroissant) selon un descripteur.
  • Filtrer : ne garder que les objets qui respectent une condition.
  • Calculer : par exemple faire une somme ou une moyenne.

Reprenons la table des livres (titre, auteur, annee, pages).

a. Si je trie par annee croissante, quel livre apparaît en premier ?

b. Si je filtre les livres de plus de 300 pages, lesquels sont gardés ?

Activité 3 — Un aperçu en Python

On peut aussi lire un CSV avec Python, grâce au module csv :

import csv
with open("livres.csv", newline="", encoding="utf-8") as f:
    table = list(csv.DictReader(f))

# Afficher le titre de chaque livre
for livre in table:
    print(livre["titre"], "-", livre["annee"])

Chaque livre est un dictionnaire : livre["titre"] donne la valeur du descripteur titre.

a. Que va afficher la boucle for de l'exemple ?

b. Dans cette table, table[0] est le premier livre. Quelle expression Python donne son année de parution ?

Activité 4 — Croiser deux tables

Quand deux tables partagent un descripteur commun, on peut les croiser. Exemple : une table eleves(nom, classe) et une table notes(nom, note) partagent le descripteur nom : on peut ainsi associer chaque élève à sa note.

a. Pour croiser une table villes(ville, pays) et une table population(ville, habitants), quel est le descripteur commun ?

Ce que je retiens

Traiter des données, c'est rechercher, trier, filtrer et calculer. On le fait facilement avec un tableur ou en Python (module csv). Lorsque deux tables partagent un descripteur commun, on peut les croiser pour combiner leurs informations.
