Séance 3: Créer une application smartphone

💾 Tes réponses sont enregistrées automatiquement sur cet appareil.

Accéder au logiciel AppInventor:

  1. Téléchargez le fichier Tri_selectif_eleve.aia

  2. Connectez-vous à l’application App Inventor : code.appinventor.mit.edu

  3. Ouvrez le fichier : Tri_selectif_eleve.aia à partir du dossier Downloads sur votre ordinateur

Importer projet de mon ordinateur

Parcourir

AppInventor + Micro:Bit

Connectez votre application à a carte Micro:bit:

Attention:

Connectez votre appareil à Micro:bit via Bluetooth (pair your devices)

Lors de l'appairage, assurez-vous de maintenir enfoncés les boutons A, B et Reset suffisamment longtemps puis enlever le bouton Resert pour que toutes les LED de l'écran s'allument.
