Synthèse : capteur, actionneur, interface

💾 Tes réponses sont enregistrées automatiquement sur cet appareil.

Structurer les connaissances — Capteur, actionneur, interface

Pour programmer un objet connecté, on utilise un système technique qui comporte des capteurs, des actionneurs et une interface.

  • Un capteur réalise l'acquisition d'une grandeur physique (température, luminosité, présence, distance…) et la transforme en signal électrique.
  • Un actionneur convertit l'énergie qu'il reçoit pour réaliser une action.
  • L'interface programmable traite les informations reçues des capteurs grâce à un programme, puis envoie des ordres pour piloter les actionneurs.

Pour que l'interface traite les données et envoie des ordres, il faut d'abord la programmer.

Ce que je retiens

Un capteur acquiert une grandeur physique. Un actionneur réalise une action à partir de l'énergie qu'il reçoit. Une interface établit la communication entre tous les composants : elle reçoit les informations des capteurs (ou de l'utilisateur), effectue des traitements et envoie des ordres aux actionneurs.
