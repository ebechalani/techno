III- Pour commencer l'application suivez les étapes suivantes:

1- Accéder à code.appinventor.mit.edu

2- Cliquez sur "continue without an account"

3- Sauvegardez le code que vous obtenez sur une feuille, pour le réutiliser à chaque fois vous voulez modifier votre application.

4- Commencez l'interface de votre application
